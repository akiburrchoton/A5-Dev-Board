Question-1: What are the different ways to select an element in the DOM?

Answer: We can select an element in different ways such as using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector, querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent?

Answer: - By the innerHTML, we can set or get the HTML content inside the element incluging tags. It also renders bold text.
- innerText only works with the available texts inside the element and it ignores hidden elements.
- Finally, textContent works with all the texts inside the element including hidden ones and HTML tags.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a process of using event propagation or bubbling so that we can handle events at a higher level in the DOM than the element on which the event is from.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a process when an event triggered on a child element and it keeps going upward through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: By using createElement() to create, appendChild() to add anything, removeChild() to remove something.
